Space Invaders

Built a retro shooter game in Python under the supervision of Prof. Walker White. The game is designed to vaguely resemble the popular space invaders arcade game from 1978. I used 8-bit images to connote the aliens and the spaceship . The game employed a round system with each round having increased alien speed. The player controls the ship and tries to eliminate all the aliens before his lives run out. I designed extra life and extra laser bolt powerups. This project relied heavily on the concept of inheritance.

Stock Exchange Simulator

A stock exchange simulator I built in Ocaml. It allows users to create portfolios using past stock information. The user can watch as the stock market changes over time. Users are allowed to set the amount of money they want to start with and the duration of the simulation. This project employed concepts such as functors and promises.
